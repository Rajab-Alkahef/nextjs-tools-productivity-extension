# Media Folder

Place your extension icon here:

- **File name**: `icon.png`
- **Size**: 128x128 pixels
- **Format**: PNG

The icon will be displayed in:
- VS Code Extensions view
- VS Code Marketplace
- Extension details page

## Quick Setup

1. Create or download a 128x128 PNG image
2. Name it `icon.png`
3. Place it in this `media/` folder
4. The extension will automatically use it

## Icon Design Tips

- Keep it simple and recognizable
- Use high contrast colors
- Avoid small text
- Test how it looks at small sizes
